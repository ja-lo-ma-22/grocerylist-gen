# Grocery List Generator

grocerylist-gen

A program that automatically generates grocery lists and saves local
data for the user.

features:

* Saves data about groceries (TODO)
	* Name
	* Price
	* Purchase History
	* How often you want to buy it

* Creates a grocery list based on this data.
* Allows manual modification
* Is friendly and helpful.
* Has an interactive mode and a CLI mode.
